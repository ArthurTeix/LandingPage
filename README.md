🌐 Landing Page Project

Este projeto consiste na criação de uma Landing Page utilizando HTML e CSS, com o objetivo de aprimorar habilidades fundamentais de desenvolvimento front-end, tanto na estruturação quanto na estilização de páginas web.

🚀 Objetivo

Desenvolver uma página moderna e responsiva, aplicando boas práticas de marcação semântica, organização de código e estilização, visando maior legibilidade, manutenção e experiência do usuário.

🛠️ Tecnologias Utilizadas

- HTML5
- CSS3

📚 Conceitos Praticados

🔹 HTML (Estrutura)

- Uso de tags semânticas (sections, header, footer, etc.)
- Criação de menus de navegação
- Implementação de formulários
- Inserção e organização de imagens
- Estruturação de tabelas

🔹 CSS (Estilização)

- Utilização de diferentes tipos de display
- Aplicação de unidades relativas como rem
- Criação de menu hambúrguer
- Uso de background images e suas formatações
- Estilização de formulários

📱 Responsividade

- Implementação de media queries
- Adaptação para diferentes tamanhos de tela (desktop, tablet e mobile)

🧩 Boas Práticas

- Modularização do código
- Organização visando legibilidade e manutenção
- Uso de comentários explicativos
